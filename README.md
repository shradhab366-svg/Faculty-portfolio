# 🎓 Faculty-Portfolio Website
## 📌 About the Project

This website showcases:

* Academic profile
* Teaching experience
* Research publications
* Certifications & awards
* Skills and expertise
* Contact information

The goal of this project is to create a clean and professional digital portfolio for academic and research purposes.

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* JavaScript
* AOS (Animation on Scroll)
* Swiper.js

---

## ✨ Features

* Responsive design (Mobile + Desktop)
* Smooth animations
* Interactive portfolio section
* Contact form integration
* Light & professional UI design
